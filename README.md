## 🚀 Overview

This website is a premium, interactive, 3D animated portfolio designed to display software engineering skills, projects, and backend expertise in an immersive universe environment.

### 🌌 Interactive Live Demo
Once deployed on Vercel, you can view the live site and interact with it.

---

## ✨ Features

- **🛸 Immersive 3D Space Background**: Stars, planets, and interactive particles powered by **Three.js** and **React Three Fiber**.
- **🎨 Framer Motion Animations**: Sleek entrance transitions, hover interactions, scroll-bound animations, and smooth component reveals.
- **🛡️ Custom Components**: Fully optimized navigation, footer, dynamic skill cards, and project showcase sections.
- **📱 Fully Responsive Design**: Designed with Tailwind CSS for seamless viewing across smartphones, tablets, and large screens.
- **🚀 SEO Optimized**: Built using Next.js App Router metadata structure with pre-rendered pages for performance.

## 🛠️ Tech Stack & Technologies

### Frontend
- **Framework**: Next.js 14 (React.js)
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **3D Graphics**: Three.js, React Three Fiber & Drei

### Backend & Databases
- **Runtime**: Node.js & Express.js
- **Databases**: MongoDB, MySQL, Firebase, Supabase

### Languages & Mobile Dev
- **Languages**: TypeScript, JavaScript, HTML5, CSS3, C++, Java
- **Mobile**: Flutter, React Native

## 📁 Folder Structure

```bash
space-portfolio/
  ├─ app/                 # Next.js page routing, layouts, and styles
  ├─ components/          # Reusable React components
  │   ├─ main/            # Global components (Navbar, Footer, Hero, Skills)
  │   └─ sub/             # Sub-components (SkillText, SkillProvider, ProjectCard)
  ├─ config/              # App config and metadata definitions
  ├─ constants/           # Dynamic data for projects, socials, and skills
  ├─ lib/                 # Shared utilities and Framer Motion helpers
  └─ public/              # Static assets (images, skills icons, project assets)
```

## ⚙️ Local Development Setup

To run this repository locally:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/AFNAN7788/Portfolio-.git
   cd Portfolio-
   ```

2. **Install dependencies:**
   ```bash
   npm install --legacy-peer-deps
   ```

3. **Run the local development server:**
   ```bash
   npm run dev
   ```
   Open `http://localhost:3000` to view it in your browser.

---

## 🌐 Deploy to Vercel

To deploy your portfolio site to Vercel:

1. Push all your changes to your new GitHub repository.
2. Sign in to [Vercel](https://vercel.com).
3. Select **Add New Project**, search/import the `Portfolio-` repository.
4. Keep the default settings (Vercel automatically detects Next.js configuration).
5. Click **Deploy**!
